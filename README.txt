== Description ==
This is plugin for posting pictures from twitter messages to the wordpress posts.

[Plugin Developers page](http://cetera.ru/ "cetera") 

== Installation ==

1. Upload twitter-repost folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure you can upload images in the media uploader.

== Frequently Asked Questions ==

= How add my stream to the site =

That's pretty simple. You need to filled out the form at the plugin's page (follow the amount recomendation) and click 'Add' button. Your tweets media data will update every our on your site. There are several fields which you can use:

 1. Username - twitter username.
 2. Amount of tweets - amount of tweets you want to check out each our.
 3. Category to post in - you can choose multiple categories.

If you do everything correctly your stream will be start to work immediatly.
